# 🎲 Dice Game

A fun and strategic dice game built with **React**, **Vite**, and **Tailwind CSS**. The objective is to make all **10 dice display the same number** within **10 attempts** by strategically freezing and rolling dice.

## 🚀 Features

* 🎲 Roll 10 dice with random values
* 🖱️ Click on a die to freeze or unfreeze it
* 🔒 Frozen dice keep their value during future rolls
* 🔟 10 attempts to complete the game
* 🏆 Win when all 10 dice show the same number within 10 attempts you won otherwise it will result in a loss
* 🔄 Start a new game with a single click
* 📱 Responsive design
  

## 🛠️ Tech Stack

* React
* Vite
* JavaScript (ES6+)
* CSS

## 🎮 How to Play

1. Click **Roll Dice** to roll all unfrozen dice.
2. Choose a number and click the dice showing that number to **freeze** them.
3. Roll the remaining unfrozen dice.
4. Continue freezing matching dice until all 10 dice display the same number.
5. You have a maximum of **10 attempts**.
6. If all dice match within 10 attempts, **you win**.
7. If you use all 10 attempts without matching all dice, **you lose**.
8. Click **New Game** to play again.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder:

```bash
cd dice-game
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
